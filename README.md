# 🎓 Student Course Registration System

A full-stack web application that allows students to register, log in, explore available courses, and enroll seamlessly through a modern and responsive interface.

---

## 🚀 Features

✨ User Authentication (Signup & Login)  
📚 View Available Courses  
📝 Enroll in Courses  
⚡ Real-time UI Updates  
🎨 Responsive & Clean Interface  

---

## 🛠️ Tech Stack

**Frontend:**
- HTML5
- CSS3
- JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB

## 🌐 API Endpoints

| Method | Endpoint | Description |
|-------|--------|------------|
| POST | /api/auth/register | Register user |
| POST | /api/auth/login | Login user |
| GET | /api/courses | Get all courses |
| POST | /api/enroll/enroll | Enroll in course |
